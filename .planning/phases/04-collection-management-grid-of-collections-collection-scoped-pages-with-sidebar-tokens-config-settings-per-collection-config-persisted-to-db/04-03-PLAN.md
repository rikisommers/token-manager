Create `src/components/CollectionCard.tsx` as a 'use client' component.

Props interface:
```typescript
interface CollectionCardProps {
  collection: CollectionCardData;  // from collection.types.ts
  onRename: (id: string, newName: string) => Promise<void>;
  onDelete: (id: string) => void;  // triggers parent dialog
  onDuplicate: (id: string) => Promise<void>;
  onClick: (id: string) => void;
}
```

Card layout (Tailwind):
- Outer: `relative bg-white rounded-lg border border-gray-200 p-4 cursor-pointer hover:shadow-md hover:border-gray-300 transition-all group`
- Card click (on the outer div, NOT the kebab area) calls `onClick(collection._id)`
- Stop propagation on kebab menu so click doesn't navigate

Kebab menu:
- Three-dot icon (MoreVertical from lucide-react) shown on group-hover: `opacity-0 group-hover:opacity-100`
- Position: absolute top-2 right-2
- Clicking the icon toggles a dropdown with three items: Rename, Duplicate, Delete
- Use simple local `isMenuOpen` state — no need for a Radix dropdown for this simple menu
- Close menu on click outside (useEffect with document click listener, check `!ref.current.contains(event.target)`)

Inline rename:
- When "Rename" selected from menu: set `isRenaming = true`, render an `<input>` in place of the `<h3>` title
- Input has the current name as value, auto-focused via `autoFocus`
- On Enter key: call `onRename(id, newName)` then `setIsRenaming(false)`
- On Escape key: `setIsRenaming(false)` (no save)
- On blur: treat same as Enter (save the value)
- Show loading state during rename (disable input while pending)

Card body:
- Title (h3): collection name, `text-gray-900 font-semibold text-base truncate pr-6`
- Description: `text-gray-500 text-sm mt-1 line-clamp-2` — show "No description" in gray-300 if null
- Tags: render as small chips `bg-gray-100 text-gray-600 text-xs px-2 py-0.5 rounded-full` — skip if empty array
- Stats row: token count (`{count} tokens`) + last modified (`Updated {relative date}`) — use `new Date(updatedAt).toLocaleDateString()`
- Integration badges row: if `figmaConfigured`, show small badge "Figma" with green dot; if `githubConfigured`, show "GitHub" badge — use `bg-green-50 text-green-700 text-xs px-1.5 py-0.5 rounded border border-green-200`

The "+ New Collection" card is handled by the parent page (not this component).
