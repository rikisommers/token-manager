<task type="auto">
  <name>Task 1: Install shadcn/ui dependencies and generate components.json</name>
  <files>package.json, components.json, src/lib/utils.ts</files>
  <action>
Install shadcn/ui and its peer dependencies, then initialize the configuration:

1. Install required packages:
   ```
   yarn add class-variance-authority clsx tailwind-merge lucide-react @radix-ui/react-dialog @radix-ui/react-select @radix-ui/react-tabs @radix-ui/react-slot
   ```

2. Create `components.json` manually (shadcn CLI requires interactive prompts; create it directly instead):
   ```json
   {
     "$schema": "https://ui.shadcn.com/schema.json",
     "style": "default",
     "rsc": true,
     "tsx": true,
     "tailwind": {
       "config": "tailwind.config.js",
       "css": "src/app/globals.css",
       "baseColor": "slate",
       "cssVariables": true,
       "prefix": ""
     },
     "aliases": {
       "components": "@/components",
       "utils": "@/lib/utils"
     }
   }
   ```

3. Create `src/lib/utils.ts`:
   ```typescript
   import { type ClassValue, clsx } from 'clsx';
   import { twMerge } from 'tailwind-merge';

   export function cn(...inputs: ClassValue[]) {
     return twMerge(clsx(inputs));
   }
   ```

Note: Do NOT use `npx shadcn-ui@latest init` — it requires interactive terminal input that Claude cannot provide. Manually wire the configuration instead.
  </action>
