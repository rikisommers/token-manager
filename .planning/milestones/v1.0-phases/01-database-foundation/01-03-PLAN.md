    Create directory `scripts/` at the project root and file `scripts/seed.ts`.

    The script must:
    1. Load environment variables from `.env.local` using `dotenv` (install `dotenv` if not already a dependency: `yarn add -D dotenv`).
    2. Call `dbConnect()` to establish the MongoDB connection.
    3. Recursively walk the `tokens/` directory (relative to project root) and collect all `.json` files.
    4. For each JSON file:
       a. Derive the collection name: use the file's path relative to `tokens/`, without the `.json` extension, with slashes replaced by ` / ` — e.g. `globals/color-base.json` → `globals / color-base`, `brands/brand1/tokens.json` → `brands / brand1 / tokens`. This gives reasonable default names that users can rename in Phase 4.
       b. Check if a collection with that name already exists in MongoDB (idempotency check using `TokenCollection.findOne({ name })`) — skip with a log message if it exists.
       c. Parse the JSON file content.
       d. Create a new TokenCollection document: `{ name, tokens: parsedJson, sourceMetadata: null, userId: null }`.
    5. After processing all files, log a summary: `Seed complete: X inserted, Y skipped (already existed)`.
    6. Disconnect from MongoDB and exit with code 0 on success, code 1 on error.

    Script should be runnable via: `npx ts-node --project tsconfig.json scripts/seed.ts`

    Add a `"seed"` script to the root `package.json`:
    ```json
    "seed": "dotenv -e .env.local -- npx ts-node --project tsconfig.json scripts/seed.ts"
    ```
    Note: if `dotenv-cli` is needed for the above shell command, use `ts-node -r dotenv/config scripts/seed.ts` instead (no extra package needed — dotenv is already installed as a dev dep, and `dotenv/config` auto-loads `.env.local`). Use whichever approach works with the existing toolchain.

    Full pattern:

    ```typescript
    import 'dotenv/config';
    import path from 'path';
    import fs from 'fs';
    import dbConnect from '../src/lib/mongodb';
    import TokenCollection from '../src/lib/db/models/TokenCollection';

    const TOKENS_DIR = path.join(process.cwd(), 'tokens');

    function walkDir(dir: string, baseDir: string): string[] {
      const results: string[] = [];
      const entries = fs.readdirSync(dir, { withFileTypes: true });
      for (const entry of entries) {
        const fullPath = path.join(dir, entry.name);
        if (entry.isDirectory()) {
          results.push(...walkDir(fullPath, baseDir));
        } else if (entry.isFile() && entry.name.endsWith('.json')) {
          results.push(fullPath);
        }
      }
      return results;
    }

    function deriveCollectionName(filePath: string): string {
      const rel = path.relative(TOKENS_DIR, filePath);
      return rel.replace(/\.json$/, '').replace(/[/\\]/g, ' / ');
    }

    async function seed() {
      await dbConnect();
      const files = walkDir(TOKENS_DIR, TOKENS_DIR);
      let inserted = 0;
      let skipped = 0;

      for (const file of files) {
        const name = deriveCollectionName(file);
        const existing = await TokenCollection.findOne({ name });
        if (existing) {
          console.log(`[SKIP] "${name}" already exists`);
          skipped++;
          continue;
        }
        const raw = fs.readFileSync(file, 'utf-8');
        const tokens = JSON.parse(raw);
        await TokenCollection.create({ name, tokens, sourceMetadata: null, userId: null });
        console.log(`[INSERT] "${name}"`);
        inserted++;
      }

      console.log(`\nSeed complete: ${inserted} inserted, ${skipped} skipped`);
      process.exit(0);
    }

    seed().catch((err) => {
      console.error('Seed failed:', err);
      process.exit(1);
    });
    ```
